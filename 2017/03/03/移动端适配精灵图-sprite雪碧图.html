移动端适配精灵图(sprite雪碧图)

发表于 2017-03-03 | 分类于 CSS | | 阅读次数

移动端精灵图可以使用按照PC端的进行移植、只是把

background-position: 1rem 3.1rem; 对应的px转换成rem即可

大体上这样适配没什么问题,如果公司要求不是很严格的话

在一些手机里面,会有一切缺陷,有些图片少了那么1px。但是本着进益求精的原则，解决方案总结如下：(使用百分比方式、需要sass的mixin进行封装)
